# TERMUX
```bash
> download termux nya link termux udah gua Taruh di install
> Buka
> termux-setup-storage
> Apt update && apt upgrade 
>  pkg install git
> git clone https://github.com/Kenzquerza/bot-gua
> CD bot-gua 
> pkg install nodejs
> pkg install ffmpeg
> npm i -g cwebp
> npm i -g ytdl && npm i && npm i got
> bash install.sh
> node index.js
```
